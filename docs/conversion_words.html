d
date
d{dateTimeFormat}
d(sub-pat)
 Date: Timestamp of the log event, supporting a multitude of formats. The format is specified in either a special sub-pattern following the strftime library function (time.h) format notation OR in the parameter dataTimeFormat which, like Logback, uses a more readable format notation similar to java.text.SimpleDateFormat. Either a sub-pattern or a conversion parameter or neither must be present; it is an error if both are.
Example: 
%d
